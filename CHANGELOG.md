# 앱 변경 기록

이 문서는 GitHub의 복잡한 커밋 목록을 대신해 중요한 변경사항만 날짜별로 정리합니다.

## 2026-07-12

- 애드센스 심사 전체 재점검 및 v35 안정판 구성
- 실제 광고가 아닌 `Google AdSense` 빈 자리 표시 제거
- 상단 탐색 메뉴와 `guide.html`, `privacy.html`, `404.html` 추가
- 사용법, 번호 생성 원리, 수동·자동 차이, 책임 있는 이용, FAQ 콘텐츠 보강
- 외부 뉴스·YouTube 자동 노출과 철학 앱 플로팅 홍보 제거
- 심사 중에는 운영자가 확인한 중립적 주제를 난수 재료로 사용
- 당첨 운·과학적 추출 등으로 오해할 표현을 오락용 무작위 번호로 수정
- 깨진 경로와 광고 코드가 있던 공개 백업 HTML·CSS 제거
- `robots.txt`와 `sitemap.xml` 추가, 철학 앱은 심사 중 검색 대상에서 임시 분리
- PWA 아이콘을 선언과 일치하는 192px·512px PNG로 수정
- 결과 화면에서 처음으로 돌아간 뒤 다시 실행되지 않던 문제 수정
- 아이폰 클립보드 복사 실패 시 대체 복사 방식과 안내 추가
- 구형 아이폰 화면 높이 대체값과 서비스워커 오프라인 캐시 일치 처리
- 애드센스 심사 중 공개 사이트에서 아이폰 QR 시험 기능을 임시로 숨김
- QR 시험 코드는 Git 기록에 보존하고 공개 심사판에서 제거
- GitHub 첫 화면에 두 앱을 구분하는 관리 안내판 추가
- FirstLott와 Philosophy Roulette의 실행 주소 및 코드 위치 정리
- `philosophy-V1`을 현재 사용 버전으로 표시
- `philosophy`를 이전 버전으로 표시
- FirstLott 첫 이슈창이 외부 뉴스 API를 기다리지 않고 즉시 열리도록 수정
- 아이폰 캐시가 외부 뉴스·광고 요청을 방해하지 않도록 서비스워커 수정
- 최신 로또 당첨정보를 제1232회(12, 15, 19, 22, 24, 36 / 보너스 3)로 갱신
- 예전 `First-Lotto` 저장소와 현재 `vokim2503.github.io` 저장소의 차이를 안내판에 표시
- `v32`, `v34`는 배포 번호가 아니라 아이폰 캐시 갱신용 내부 버전임을 기록

## 2026-07-11 — FirstLott

- 아이폰에서 흔들리던 번호 선택 화면을 안정적인 페이지 방식으로 변경
- 1부터 45까지 모든 번호를 3페이지로 확인할 수 있도록 개선
- 숫자 숨김·숫자 표시 수동 모드 추가
- 자동 5게임 생성 및 번호 등장 애니메이션 추가
- 최신 1231회 당첨번호 표시 및 자동게임 비교 기능 추가
- QR 당첨 확인 기능과 개인정보 안내 추가
- QR 촬영 사진의 여러 영역을 판독하는 기능 추가
- QR에서 읽은 복권 번호와 당첨번호의 일치 표시 화면 추가

> 아이폰 QR 판독 기능은 기기와 브라우저에 따라 동작이 다를 수 있어 계속 점검 중입니다.

- Philosophy Roulette 제작 및 `philosophy-V1` 폴더에 배포
- GitHub Pages 공개 주소 연결
- Google Analytics 및 AdSense 관련 코드 연결

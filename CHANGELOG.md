# 앱 변경 기록

이 문서는 GitHub의 복잡한 커밋 목록을 대신해 중요한 변경사항만 날짜별로 정리합니다.

## 2026-07-12

- GitHub 첫 화면에 두 앱을 구분하는 관리 안내판 추가
- FirstLott와 Philosophy Roulette의 실행 주소 및 코드 위치 정리
- `philosophy-V1`을 현재 사용 버전으로 표시
- `philosophy`를 이전 버전으로 표시

## 2026-07-11 — FirstLott

- 아이폰에서 흔들리던 번호 선택 화면을 안정적인 페이지 방식으로 변경
- 1부터 45까지 모든 번호를 3페이지로 확인할 수 있도록 개선
- 숫자 숨김·숫자 표시 수동 모드 추가
- 자동 5게임 생성 및 번호 등장 애니메이션 추가
- 최신 1231회 당첨번호 표시 및 자동게임 비교 기능 추가
- QR 당첨 확인 기능과 개인정보 안내 추가
- QR 촬영 사진의 여러 영역을 판독하는 기능 추가
- QR에서 읽은 복권 번호와 당첨번호의 일치 표시 화면 추가

> 아이폰 QR 판독 기능은 기기와 브라우저에 따라 동작이 다를 수 있어 계속 점검 중입니다.

## 이전 작업
